### johnhib.com — Subhead Shine Effect (9:21–9:24 PM)

**Feature:** On hover, a light beam sweeps left→right across the subhead text (sunrise/diamond shine).

**Implementation:**
- CSS keyframe `text-shine` in `globals.css`: gradient sweep via `background-position` animation
- Gradient: muted → gold (`#C8A96E`) → warm white (`#F5F2EC`) → gold → muted; band is ~12% wide
- Background-clip text trick: `-webkit-background-clip: text`, `-webkit-text-fill-color: transparent`
- Class `.subhead-shine` applied on `mouseenter`, removed on `mouseleave`
- Animation restarts each hover via `el.classList.remove` → `void el.offsetWidth` (force reflow) → `el.classList.add`
- Duration: `1.3s ease forwards` — one pass, then stops
