## What We Built Today

### 1. OG Image Pipeline (johnhib.tech)
- **Problem**: iMessage link previews showed no rich card — just domain + Safari compass icon
- **Root cause**: Dynamic `/og?slug=...` edge route fetched fonts from same domain (circular call in Apple's crawler)
- **Solution**: Pre-generate static PNGs at build time using `satori` + `@resvg/resvg-js`
- **Script**: `scripts/generate-og.mjs` — reads article data, generates `public/og/[slug].png`
- **Build hook**: `"build": "node scripts/generate-og.mjs && next build"`
- **Metadata**: Updated to point to static PNGs (`/og/home.png`, `/og/${slug}.png`)
- **Result**: iMessage now renders full banner card
