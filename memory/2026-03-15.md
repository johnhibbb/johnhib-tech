### Chainline issue (diagnosed from photos)
- Problem: 5DEV 8-bolt direct mount chainring sitting too far inboard
- B-gap adjusted to 7mm (SRAM spec) — helped large cogs, small cogs still noisy
- Root cause: chainline too far inboard — chain cross-chaining severely to small cogs
- Fix: BSA spindle spacers on non-drive side to push crankset outboard ~2-3mm to 49mm target
- Cannot use chainring bolt spacers — 8-bolt direct mount has no bolt interface for this
- Recommendation: Take to bike shop Monday (Safety Cycle ~3-5mi, or Bicycle Kitchen ~1-2mi from 90038)
- Script for shop: "SRAM Force AXS 1x12, 5DEV 8-bolt direct mount 42T on BSA threaded BB. Chainline too far inboard. Need BSA spacers on non-drive side to reach 49mm chainline."

## DJI Osmo 360 / ProRes Workflow

### Problem
DJI Studio only exports H.264 — no ProRes path. OSV format is fully proprietary, no open source parser exists.

### Recommended workflow
1. DJI Studio: all 360 reframing + virtual camera keyframes (purpose-built, don't skip this)
2. Export highest quality H.264 from Studio
3. DaVinci Resolve: edit + color grade all footage (360 export + Action 5 Pro)
4. Export final as ProRes 422 HQ
